reddplusid.mission Installation
-------------------------------

To install reddplusid.mission using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``reddplusid.mission`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        reddplusid.mission

* Re-run buildout, e.g. with:

    $ ./bin/buildout
